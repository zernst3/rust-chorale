# leptos-with-selection

Leptos example demonstrating per-row checkboxes, select-all, and reactive
selection count.

## Running

```sh
# From this directory:
trunk serve --open

# Or from the workspace root:
trunk serve --open --package leptos-with-selection
```

Requires [Trunk](https://trunkrs.dev): `cargo install trunk`

> **Note:** This is a Leptos/WASM example. Do **not** use `dx serve` (that is for
> Dioxus examples). Use `trunk serve` as shown above.
