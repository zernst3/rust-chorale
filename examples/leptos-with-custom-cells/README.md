# leptos-with-custom-cells

Leptos example demonstrating `RenderKind::Badge` and `CellRenderers` for
custom cell markup.

## Running

```sh
# From this directory:
trunk serve --open

# Or from the workspace root:
cd examples/leptos-with-custom-cells && trunk serve --open
```

Requires [Trunk](https://trunkrs.dev): `cargo install trunk`

> **Note:** This is a Leptos/WASM example. Do **not** use `dx serve` (that is for
> Dioxus examples). Use `trunk serve` as shown above.
